Gallery 2

Simple img example using the bootstrap column layout (col-sm-4 classes). The first bit indicates column, second indicates the screen size it will change at (xs, sm, md, lg), and the last number indicates the width, each row adds to 12.

The css :hover property is used to add animation, first row using scale(1.1), the second using margin-top:-5px;. The top row also has text that will appear when the cursor is left ontop of the image. Adjust the padding property in the CSS to adjust the spacing between the images.
